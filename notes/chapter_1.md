# 1. By the C, by the C, by the Beautiful C
- C is a high-level programming language like other languages you might know.
- It's an imperative and a procedural programming language, which means that a C program is expressed as a sequence of statements (steps) for the computer to execute and that C programs are structured as a set of functions (procedures).
- The C programming language is less abstracted from the computer's machine language than some other languages with which you might be familiar with.
- C doesn't have support for object-oriented programming or have a rich set of high-level programming abstractions.
- C programmers retain more control over how their programs execute on the hardware, and they can write code that runs more efficiently than equivalent code written using the higher-level abstractions provided by other programming languages.
  - In particular, they have more control over how their programs manage memory, which can have a significant impact on performance.
    - Thus, C remains the de facto language for computer systems programming where low-level control and efficiency are crucial.
